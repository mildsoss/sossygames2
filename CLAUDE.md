# Development Workflow

**Always use `bun`, not `npm` or `yarn`.**

```sh
# 1. Make changes
# 2. Typecheck before completing
bun run typecheck
# 3. Lint and format before completing
bun run check
```

# Tech Stack

- Framework: Astro
- Language: Astro w/ TypeScript
- Astro Plugins: SolidJS
- Styling: TailwindCSS
- Package Manager: Bun
- Formatter & Linter: Biome
